# queue.Last()
queue.Peek() 와 반대로 Last 는 마지막요소를 반환

<div class="notice--primary" markdown="1"> 

```c# 
else if (a[0] == "front")
{
    if (queue.Count > 0)
    {
        sb.AppendLine($"{queue.Peek()}");
    }
    else
    {
        sb.AppendLine("-1");
    }
}
else if (a[0] == "back")
{
    if (queue.Count > 0)
    {
        sb.AppendLine($"{queue.Last()}");
    }
    else
    {
        sb.AppendLine("-1");
    }
}

```
