# 정렬

<div class="notice--primary" markdown="1"> 

```c# 

List<string> list = words.ToList();

list.Sort((x, y) =>
{
    if (x.Length == y.Length)
    {
        return x.CompareTo(y);
    }
    else
    {
        return x.Length.CompareTo(y.Length);
    }
});

```

- hashset -> 중복제거
