#   4. 내용 접기/펼치기)    
마크다운에선 지원하지 않고 HTML의 details 태그로 사용 가능하다. ```div markdown=”1”``` 은 jekyll에서 html사이에 markdown을 인식 하기 위한 코드이다.  
```
<details>
<summary>접기/펼치기</summary>
<div markdown="1">       
 ⭐⭐⭐⭐ 깔깔 ⭐⭐⭐⭐
</div>
</details>
```  
<details>
<summary>접기/펼치기</summary>
<div markdown="1">       
 ⭐⭐⭐⭐ 깔깔 ⭐⭐⭐⭐
</div>
</details>
